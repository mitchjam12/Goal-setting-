# Waypoints

Waypoints is a small goal-tracking app. You can add a goal, set a category and target date, add milestones, and tick milestones off as you make progress.

## How to Run It

Open `index.html` in a browser.

## Files

- `index.html` builds the page.
- `style.css` controls how the app looks.
- `script.js` stores goals in the browser and updates the page.

## Demo Ideas

Good first changes for this project:

- Add a new goal category.
- Change the colours in `style.css`.
- Add a button to clear completed goals.
- Add a short explanation to the empty state.
